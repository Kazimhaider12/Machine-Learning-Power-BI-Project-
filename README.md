# Machine-Learning + Power-BI-Project-

🚦 Machine Learning Project: Road Accident Risk Prediction System

Thrilled to share my recent end-to-end Machine Learning project focused on predicting accident risk using real-world data.

🔹 Project Overview
The goal was to design a predictive system that analyzes environmental and road-related factors to estimate accident risk and provide actionable insights for safer decision-making.

🔄 End-to-End Project Flow
1️⃣ Data Source

CSV dataset containing accident-related attributes: weather, road surface, visibility, wind speed, rainfall, and more.

2️⃣ Data Preprocessing

Data cleaning (handling missing values & inconsistencies)

Encoding categorical variables

Feature engineering (risk-based features)

Train/Test split for robust evaluation

3️⃣ Model Training & Comparison

XGBoost – 0.87430 (Best Performance)

Random Forest – 0.86032

Gradient Boosting – 0.84654

KNN – 0.75743

Logistic Regression – 0.63480

➡️ XGBoost outperformed all other models and was selected as the final model.

4️⃣ Model Selection & Saving

Final model saved as .pkl

Processed dataset saved as .csv

Ensuring reproducibility and deployment readiness

5️⃣ Deployment & Business Integration

Streamlit App: Interactive prediction UI, real-time risk estimation

Power BI Dashboard: Data visualization, risk analysis, actionable business insights
